# 안산 글로벌 클라우드센터 — 시공 실적 대시보드

CHK LIST(부재 진척) + 그리드 마스터(설치 위치) 두 엑셀 파일을 **입결(입출력 결과) 데이터 표준**으로 정의하고, 시공 실적을 실시간 시각화하는 웹 대시보드.

## 빠른 시작

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

브라우저가 자동으로 열리지 않으면 http://localhost:8501 접속.

## 입결 데이터 표준 요약

### 파일 ① CHK LIST (부재 단위 마스터 + 공정 진척)

- 시트: `CHK LIST`, 데이터 시작 행: 8
- 핵심 키: **부재명(P열)**
- 7단계 공정: **취부 → 용접 → 마감 → 검사 → 전처리 → 도장 → 발송**
- 각 단계마다 *계획일 / 실적일 / 수량 / 중량* 4컬럼 묶음 (전처리만 3개)
- 현장 시공 실적: CL~CN 컬럼 (일자/수량/중량)

### 파일 ② 그리드 마스터 (부재명 ↔ GRID 매핑)

- 시트: `Sheet1`, 데이터 시작 행: 2
- 핵심 키: **DWGNO.(A열) ↔ GIRD(H열)**
- GIRD 형식: `3-5/G-H` (X열 범위 / Y열 범위)

자세한 명세와 빈 양식은 대시보드의 `📥 데이터 표준` 페이지에서 다운로드 가능.

## 주요 KPI

| KPI | 의미 |
|-----|------|
| 총 부재(행) / 총 중량(t) / 도장면적(㎡) | 전체 규모 |
| 단계별 진척률 (수량 / 중량) | 7단계 + 시공 실적 |
| 그리드 매칭률 | CHK LIST와 그리드 마스터의 join 성공률 |
| 절주별 / 팀별 / ITEM별 진척 | 다차원 분석 |
| 일자별 누계 | 시공/발송 진척 추이 |
